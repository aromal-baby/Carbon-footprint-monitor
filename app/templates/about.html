About EcoTrack

EcoTrack is a carbon footprint monitoring tool designed to help individuals and organizations track, analyze, and reduce their environmental impact. This project was developed as part of the M602 Computer Programming course at GISMA University of Applied Sciences.

How It Works

Users can input data about their energy usage, transportation, waste, food choices, and product purchases. The application calculates their carbon footprint using industry-standard emission factors and generates comprehensive reports with personalized reduction strategies.

Technologies Used

  • Python
  • Flask web framework
  • Matplotlib for visualizations
  • Pandas for data analysis
